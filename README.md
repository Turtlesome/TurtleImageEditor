# TurtleImageEditor

Available features are:
1.  Loading image
2.  Clearing the view
3.  Saving the image
4.  Changing brightness
5.  Changing contrast
6.  Changing hue
7.  Changing saturation
8.  Resizing image
9.  Rotating image
10. Mirroring image
11. Grayscale
12. Changing RGB values of an image
